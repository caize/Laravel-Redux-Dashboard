## install

```
git clone https://github.com/ncitycode/Laravel-Redux-Dashboard
cd Laravel-Redux-Dashboard
composer install
yarn install
```

## run

modify `.env` set about `email` and `db` options

```
php artisan serve
yarn dev
```

visit `localhost:8000`
